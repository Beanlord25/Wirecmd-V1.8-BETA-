# Wirecmd

This is the first public release of "Wirecmd", a little project of mine that can be used to run simple commands using the expression 2 system. This is still in development and might break if used incorrectly. The script also requires propcore to be enabled.


[![Typing SVG](https://readme-typing-svg.demolab.com?font=Roboto&weight=900&size=30&duration=3000&pause=3000&color=F70000&background=FFFFFF00&center=true&vCenter=true&width=1245&lines=WARNING:+Bugs+are+to+be+expected.+This+code+is+still+in+BETA.)](https://git.io/typing-svg)



Commands are structured to be simmilar in syntax to Minecraft commands,

## Examples
Help Command

```javascript
/help
```
Man Command

```javascript
/man [argument]
/man man
```


## Performance

### Benchmarks

| Operation | Time | Memory |
|-----------|------|--------|
| Initialize | ~3s | 1 ops 0% |
| Standard Commands | 1ms | 2 ops 0% |
| Heavy commands | 1ms | ~5 ops 0% |

### Usage Tips

1. **Avoid** using incorrect usernames for targeted commands.
2. **Know** your target when using jail commands.


### Advanced Features

- Anti-Dupe tools to prevent script kiddies.
- Jail and Tp commands if needed
- Anti tamper functions with hash code checking(COMING SOON)
- Better chip hiding (COMING SOON)
